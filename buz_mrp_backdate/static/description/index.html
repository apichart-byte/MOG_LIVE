Manufacturing Backdate/Forcedate

Allow backdating for Manufacturing Orders with full tracking

Features

  • ✅ Add Backdate and Remark fields to Manufacturing Orders
  • ✅ Wizard interface for easy backdate setting
  • ✅ Automatically update Stock Moves dates
  • ✅ Update Inventory Valuation dates
  • ✅ Update Journal Entries dates (automated valuation)
  • ✅ Track remarks across all related documents
  • ✅ Proper access rights for MRP managers
  • ✅ Compatible with Odoo 17

How it Works

1. Open a Manufacturing Order
2. Click "Set Backdate" button
3. Enter backdate and remark
4. Choose to set only, confirm, or mark as done
5. System automatically updates all related records

Use Cases

Backdating Production

When you need to record manufacturing activities that happened in the past with proper documentation.

Audit Trail

Maintain complete audit trail with remarks explaining why backdating was necessary.
